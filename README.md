# toon-star-rader
업로드 된 웹툰의 별점 테러 모니터링 시스템

## 목적
작품별, 일자별 별점 태러가 발생하는 시간과 별점 태러를 진행한 유저정보를 실시간으로 모니터링 하는것

## 스팩과 기준은 오프라인때 공유드리려고 합니다!


## 요구사항
- Java
- Spring boot 2.x
- Message Queue (kafka) 
- DB(Mysql, mongodb, redis) 
- 간단한 FrontEnd (graph 표현)
